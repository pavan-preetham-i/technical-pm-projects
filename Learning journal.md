# Day 1 – Git & GitHub

## What I learned

- Git stores versions of code.
- GitHub hosts repositories.
- Local and remote repositories are different.
- Commits are snapshots.
- Push uploads commits.
- Pull downloads commits.
- Merge conflicts happen when the same file changes.

## Biggest confusion

Understanding the difference between Git and GitHub.

## Biggest realization

Every project has its own Git repository.
